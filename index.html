Rent-My-Pet

rent-my-pet.com is a website that allows users to rent out their pets to people who are interested in taking care of a cat or dog for a day. The website was built using Ruby on Rails, Backbone.js, Bootstrap, HTML, CSS, JQuery UI, and Google Maps API. Users can filter selections by animal type, location, color, gender, and age.

Trello Clone

TrelloMe is a clone of Trello.com. With this convenient task manager, users can create boards, lists, and cards. The Trello cone was built using Ruby on Rails, Backbone.js, Bootstrap, and JQuery UI.
